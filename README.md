# LubiÄ™ Placki

Website dedicated to sharing recipes - backend part

## Installation

Download or clone repository

Add .env file

```
DATABASE_URI=...
PORT=...
ACCESS_TOKEN_SECRET=...
REFRESH_TOKEN_SECRET=...
JWT_SECRET=...
```

Run

```bash
# installing dependances
$ yarn install
```

## Running the app

```bash
# development
$ yarn start
```
